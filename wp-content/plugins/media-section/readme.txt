Adds a new "Media" (Медиа) section that fully duplicates the blog functionality.

== Description ==

Этот плагин создаёт новый раздел «Медиа», который полностью копирует
функционал блога на сайте:

* **Та же вёрстка страниц.** Раздел регистрируется как пользовательский тип
  записи `media`. Благодаря иерархии шаблонов WordPress одиночные страницы
  медиа используют тот же `single.php` активной темы (тот же Brizy-шаблон, что
  и у постов блога), а архив `/media/` — тот же `archive.php`. Отдельные
  файлы шаблонов не нужны — вёрстка идентична блогу.
* **Те же функции создания статей.** Тип записи `media` повторяет набор
  возможностей встроенного типа `post`: заголовок, редактор (включая
  Gutenberg через REST), автор, изображение записи, цитата, комментарии,
  произвольные поля, ревизии и форматы записей.
* **Рубрики и метки.** Регистрируются отдельные таксономии `media_category`
  (как рубрики) и `media_tag` (как метки), чтобы раздел был независим от
  блога, но обладал той же системой категоризации.
* **Поддержка Brizy.** Тип `media` добавляется в список поддерживаемых Brizy
  типов записей через фильтр `brizy_supported_post_types`, поэтому медиа
  редактируются конструктором так же, как посты.

== Installation ==

1. Загрузите папку `media-section` в каталог `/wp-content/plugins/`.
2. Активируйте плагин на странице «Плагины» в админке WordPress.
3. После активации правила перезаписи (permalinks) обновляются автоматически,
   раздел доступен по адресу `/media/`.
